# 🏡 Real Estate Management System

A command-line based Real Estate Management System built using **Python**.  
The application allows clients to browse properties, search based on location, type, and price range, book properties, and manage bookings.  
Administrators can manage property listings and client records.

---

## 🚀 Features

### 👤 Client Features
- Client registration and secure login
- Search properties by location, type, and price range
- View property details
- Compare two properties
- Book and cancel property bookings
- Suggest properties within a given budget
- Change account password

### 🛠 Admin Features
- Admin login system
- Add new property listings
- Delete property records
- View registered clients
- View client booking history
- Add or remove clients
- Manage property bookings

### 👀 Guest Features
- Browse property listings
- Search and compare properties
- View suggested properties within budget
